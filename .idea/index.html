🔧 Plataformas de Control de Versiones

Aprendiendo sobre GitHub, GitLab y Bitbucket

GitHub

GitHub es una plataforma de desarrollo colaborativo para alojar proyectos utilizando el sistema de control de versiones Git.

  • 🌐 Sitio web: github.com
  • 👥 Ideal para código abierto y comunidades grandes
  • 🔒 Permite repositorios públicos y privados
  • ⚙️ Integra herramientas como Actions, Pages y Copilot

GitLab

GitLab es una plataforma DevOps completa, que ofrece repositorios Git, integración continua y herramientas de despliegue.

  • 🌐 Sitio web: gitlab.com
  • 💡 Gran enfoque en CI/CD
  • 🏢 Popular en entornos empresariales
  • 🛠️ Puede instalarse en servidores propios (self-hosted)

Bitbucket

Bitbucket es una plataforma de gestión de código fuente de Atlassian, integrada con herramientas como Jira y Trello.

  • 🌐 Sitio web: bitbucket.org
  • 👨‍💼 Ideal para equipos de desarrollo que usan herramientas Atlassian
  • 🔐 Soporta repositorios privados ilimitados (con cuenta gratuita)
  • 📦 Permite Git y Mercurial (ya no activo desde 2020)
